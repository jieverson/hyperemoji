
# Hyperemoji

Extension for emoji :keywords: in Hyper.

Check [emoji cheat sheet](https://www.webpagefx.com/tools/emoji-cheat-sheet/).

## How to use

Install [Hyper](https://hyper.is) and add `hyperemoji`
to `plugins` in `~/.hyper.js`.
